Power of Awe —

The Power of Awe is a visual mapping project that deconstructs Google Maps visually and functionally. Google Maps API was used, and geolocations were extracted from photographs onto the map.

My Creative Cartography class got the opportunity to visit and stay overnight at Santa Cruz Island to map an unfamiliar and inaccessible space. Coordinates on the map reflect moments when I was awestruck by nature and/or felt strong emotions.

Objective —

To create an interactive art pience so viewers can relive my experiences and check out the very specific coordinates of when I did what at Santa Cruz Island

Process —

I styled the Google Map used as the background with the help of these two guides created by Google: Maps Javascript API & Google Maps styling. Coordinates were extracted from photos captured by my iPhone 6s through exif2geoson. A GeoJSON file is then created. Once the webpage loads the GeoJSON file, the locations are shown as red markers. When one of the markers is clicked on, an info window pops up with some texts describing my experience and a photo.
