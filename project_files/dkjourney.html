My DK Journey is an interactive mobile app created to document my memories as an exchange student in Copenhagen, Denmark. The app uses the geolocation function to acquire the user's current location. Once the user's location is determined and when they are in a certain parameter (or when they enter a certain geofence parameter), a gif and a few simple words will appear to describe a specific memory I had during my time abroad.

Determining Where to Map â€”

The parameter of this interactive app is the UCSB campus. I mapped the city of Copenhagen on UCSB's map to display overlapping locations where I felt were significant to me. Specific coordinates were then gathered by using Google Maps.
